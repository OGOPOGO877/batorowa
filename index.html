テスト版の特徴

  • 1920×1080 画面サイズで快適
  • 中央に静止NPC 4体（HP違いでバランス確認しやすい）
  • 上部に武器・防具を等間隔配置 → Spaceで拾って即切り替え
  • F8 でデバッグビジュアルフルオン（青視界扇・水色聴覚円・黄色足音・赤銃声円・状態テキスト）
  • F9 で即リスタート（デバッグ時）
  • 音量調整UI付き（Master全体 + 自分の足音 / 自分の銃声だけ別調整）
  • ローカルサーバー必須で本物の音（walk.wav / shoot.wav）が鳴る
  • マウス視界コーン + 壁LOS + NPC索敵AI（音で反応）

主な操作

WASD / 矢印：移動
マウス：視界方向
左クリック：射撃
Space：近くのアイテム拾得（自動拾いはなし）
F8：デバッグ表示トグル
F9：リスタート（デバッグON時）
画面上部の [-] [+]：音量調整
（自分の聞こえる足音・銃声・全体）

重要：音を鳴らすには ローカルサーバー が必要です。
直接 test.html をダブルクリック（file://）すると音が出なかったり固まったりします。

GitHub Pages で見る場合

このリポジトリを GitHub Pages で公開すると、同じリンクで遊べます。
公開URL例：https://あなたのユーザー名.github.io/リポジトリ名/test.html
（index.html はこのランディングページになります）

ローカルと同じく、音を聞くときは自分のPCでローカルサーバーを立てて開くのが確実です。

ローカルで動かす方法（おすすめ）

  1. このフォルダを任意の場所に置く
  2. ターミナル / PowerShell でフォルダに移動
  3. 簡単なサーバーを起動（例）：
    • python -m http.server 8000
    • または VS Code の「Live Server」拡張
    • または npx serve .
  4. ブラウザで http://localhost:8000/test.html を開く

file:// 直開きは避けてください（特に音周り）。
